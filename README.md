# jukugo-rebulding
プレイはこちらから
(https://akapikumindayo.github.io/jukugo-rebulding/)

概要
お題の熟語を作るゲームです。
お題の熟語の読みが出るので漢字を並び替えてお題の熟語を作りましょう。

機能
「回答する」で提出をします。
熟語帳で正解した熟語を見ることができます。
熟語帳では、お気に入りと検索の機能を使えます。
一応スマホでもできるようになってます。

注意点
多分バグがたくさんあります。
Geminiと一緒に作りました。
当たり前だけど日本語にのみ対応しています。
ジャンジャン動画などに使用してもらってOKです。
だけど自分が作ったとかいうのはダメだよ。
